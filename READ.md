# Smart Healthcare Symptom Checker

A production-ready React application that helps users identify potential health conditions based on their symptoms.

## ✨ Features
- **Symptom Search:** Real-time filtering for conditions like Fever, Cold, etc.
- **Medical Database:** Built-in symptoms and precautions for common ailments.
- **Context API:** Global state management for Favorites and Dark Mode.
- **Performance:** Optimized with Debouncing and React Memoization.
- **Responsive UI:** Fully compatible with Mobile and Desktop.

## 🛠️ Tech Stack
- React (Vite)
- Context API & useReducer
- JavaScript (ES6+)
- CSS Modules / Custom Styling
